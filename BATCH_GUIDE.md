# 批量转换使用指南
## 🚀 批量转换功能

现在您的docx转markdown转换器支持批量转换功能！当双击exe文件或直接运行程序（不带参数）时，会自动批量处理目录中的所有docx文件。

## 📁 目录结构要求

```
项目目录/
├── doc/              # 📂 放入要转换的docx文件
│   ├── file1.docx
│   ├── file2.docx
│   └── file3.docx
├── output/           # 📂 转换后的markdown文件（自动创建）
│   ├── file1.md
│   ├── file2.md
│   └── file3.md
├── index.exe         # 🎯 Windows可执行文件
└── index             # 🎯 macOS/Linux可执行文件
```

## 💡 使用方法

### 方法1: 双击运行（推荐）
1. 将要转换的docx文件放入 `doc` 文件夹
2. 双击 `index.exe`（Windows）或 `index`（macOS/Linux）
3. 程序会自动批量转换所有文件到 `output` 文件夹

### 方法2: 命令行运行
```bash
# 批量转换
./index.exe          # Windows
./index              # macOS/Linux

# 或使用node运行
node index.js
```

### 方法3: 单文件转换
```bash
# 转换单个文件
./index.exe document.docx
./index.exe document.docx -o output.md
```

## 🎯 转换流程

1. **扫描文件**：程序自动扫描 `doc` 目录下的所有 `.docx` 文件
2. **创建目录**：如果 `output` 目录不存在，自动创建
3. **批量转换**：依次转换每个文件，保持原文件名
4. **显示进度**：实时显示转换进度和结果
5. **统计报告**：显示成功和失败的数量统计

## ⚠️ 注意事项

- **目录要求**：必须存在 `doc` 目录，程序会自动创建 `output` 目录
- **文件格式**：只转换 `.docx` 格式文件，其他格式会被忽略
- **文件名**：转换后保持原文件名，仅更改扩展名为 `.md`
- **中文支持**：完全支持中文文件名和内容
- **覆盖提示**：如果目标文件已存在，会被覆盖
